# 修复导航栏被顶部横幅遮挡问题

## 1. 问题分析
- FluentWindow 已经内置了 NavigationView 和 NavigationBar
- 当前 main.qml 中直接添加了覆盖整个窗口的 Column 布局，导致与内置导航栏冲突
- 视觉层级问题导致导航栏被遮挡

## 2. 修复方案

### 2.1 调整 main.qml 布局结构
- 移除直接覆盖整个窗口的 Column 布局
- 利用 FluentWindow 内置的 navigationView 属性
- 调整顶部横幅位置，避免与导航栏重叠

### 2.2 优化 NavigationBar 属性
- 确保 NavigationBar 的 z-index 属性足够高
- 调整 NavigationBar 的定位和尺寸
- 确保其始终显示在视觉层级的最上层

### 2.3 重新设计界面布局
- 移除可能导致遮挡的元素
- 调整元素间的间距和定位
- 确保导航功能正常可用

## 3. 实施步骤

- 移除直接覆盖窗口的 Column 布局

### 3.2 调整 NavigationBar 属性
- 确保 z-index 足够高
